## ğŸ› ï¸ Development

### Prerequisites
- [Quarto](https://quarto.org/docs/get-started/) installed
- Modern web browser for preview

### Local Development
```bash
# Clone the repository
git clone https://github.com/Revathyvenugopal162/cv.git
cd cv

# Preview the website locally
quarto preview

# Build the website
quarto render
```

The website will be available at `http://localhost:4200` for preview.

### Deployment
The website is automatically deployed to GitHub Pages when changes are pushed to the main branch.

## ğŸ“ Content Updates

### Adding New Blog Posts
1. Create a new `.qmd` file in `src/posts/`
2. Add appropriate frontmatter with title, description, date, and categories
3. Update `_quarto.yml` to include the new post in the sidebar navigation
4. Run `quarto render` to build

### Updating Homepage Content
- Edit `src/index.qmd` to modify homepage content
- Update sections like experience, skills, or projects
- Maintain the existing structure for consistency

### Styling Changes
- Modify `styles.css` for visual changes
- Add new CSS classes for additional components
- Test changes with both light and dark themes
